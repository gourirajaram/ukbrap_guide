`ls` gives the list of files in your current working directory. Example `/gpfs/data/mostafavilab/gouri/` <br>
`cd` sets working directory (a folder within your cluster/local storage) <br>
# _dx_ prefix to operate on files in your dnanexus project <br>
Example : 
- `dx ls` gives list of files in your UKB-RAP project <br>
- `dx mkdir` creates new directory <br> 
- [`dx upload`](https://documentation.dnanexus.com/user/objects/uploading-and-downloading-files/small-sets-of-files/uploading-using-dx) upload files from working directory to ukb-rap project. Run `dx upload filepath ` <br>
- [`dx download`](https://documentation.dnanexus.com/user/objects/uploading-and-downloading-files/small-sets-of-files/downloading-using-dx) download files from ukb-rap project to working directory. Run `dx download filepath/file ID ` <br>
