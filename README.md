# ukbrap
This repository contains scripts and the description of tools to work with [UK Biobank data](https://www.ukbiobank.ac.uk/enable-your-research/about-our-data) using the [UKB Research Analysis Platform](https://ukbiobank.dnanexus.com/landing) <br>
- `tools` - starting up various tools on UKB-RAP
- `hpc_or_local` - accessing folders, and running shell scripts on UKB-RAP through your local/bigpurple terminal <br>
